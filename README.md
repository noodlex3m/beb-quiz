# 🏛️ БЕБ Тренажер (Quiz App)

Інтерактивний веб-додаток тестових завдань для тестування з метою визначення рівня знань законодавства України кандидатів на зайняття вакантних посад у Бюро економічної безпеки України та оприлюднення зразків тестових завдань для тестування з метою визначення рівня загальних здібностей.

## Джерела

- [Про затвердження Переліку тестових завдань для тестування з метою визначення рівня знань законодавства України кандидатів на зайняття вакантних посад у Бюро економічної безпеки України та оприлюднення зразків тестових завдань для тестування з метою визначення рівня загальних здібностей](https://esbu.gov.ua/npas/pro-zatverdzhennia-pereliku-testovykh-zavdan-dlia-testuvannia-z-metoiu-vyznachennia-rivnia-znan-zakonodavstva-ukrainy-kandydativ-na-zainiattia-vakantnykh-posad-u-biuro-ekonomichnoi-bezpeky-uk)

## 🛠 Технології

- **Фреймворк:** [React](https://react.dev/) (JavaScript)
- **Збірка:** [Vite](https://vitejs.dev/)
- **Стилізація:** Звичайний CSS (Vanilla CSS)
- **Деплой:** Netlify (заплановано)

## 🏗 Архітектура компонентів

Додаток має лінійну структуру без складних стейт-менеджерів, ідеально підходить для розуміння базових хуків React (`useState`, `useEffect`).

```text
src/
 ├── assets/          # Зображення, іконки
 ├── components/      # React-компоненти
 │    ├── Header.jsx       # Шапка додатку
 │    ├── Quiz.jsx         # Головний контейнер логіки тесту
 │    ├── ProgressBar.jsx  # Індикатор прогресу (напр. 3/30)
 │    ├── QuestionCard.jsx # Відображення питання та кнопок відповідей
 │    └── ResultScreen.jsx # Екран результатів після завершення
 ├── data/            # Локальні бази даних
 │    └── questions.json   # База питань у форматі JSON
 ├── App.jsx          # Кореневий компонент (управління станами екранів)
 └── index.css / App.css # Глобальні та локальні стилі
```
