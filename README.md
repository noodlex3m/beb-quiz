# 🏛️ БЕБ Тренажер (Quiz App)

Інтерактивний веб-додаток для тестування та підготовки кандидатів на зайняття вакантних посад у Бюро економічної безпеки України. Додаток допомагає ефективно запам'ятовувати правильні відповіді завдяки миттєвому візуальному фідбеку та рандомізації варіантів.

🌐 **Live Demo:** [https://quiz.serh.one/](https://quiz.serh.one/)

## ✨ Основний функціонал

- **Миттєвий фідбек та Пояснення:** Підсвічування правильних та неправильних відповідей із розгорнутим поясненням (чому саме така відповідь правильна) після кліку. Перехід до наступного питання контролюється користувачем.
- **Робота над помилками:** На екрані результатів формується деталізований список допущених помилок, а в "Особистому кабінеті" є можливість запустити окремий іспит виключно за тими питаннями, в яких користувач помилявся раніше.
- **Офіційна симуляція іспиту:** Алгоритм випадково вибирає рівно 100 питань із загальної бази (згідно з офіційними вимогами БЕБ), щоразу генеруючи унікальний тест. Варіанти відповідей також надійно рандомізуються.
- **Особистий кабінет (Auth):** Авторизація через Google, збереження історії тестів у хмарі, підрахунок середнього балу та виявлення топ-3 "слабких" тем кандидата.

## 🏗️ Архітектура та Оптимізація (Jamstack)

Проєкт використовує гібридну архітектуру для забезпечення максимальної швидкості та уникнення лімітів на читання бази даних:

- **Статичні дані (БД питань):** Усі 884 питання зберігаються та обробляються локально у форматі `JSON`. Це гарантує нульову затримку при завантаженні іспиту та відсутність витрат на Cloud-квоти (0 reads per quiz load).
- **Динамічні дані (BaaS):** Firebase (Auth + Firestore) використовується виключно для керування сесіями користувачів та збереження персонального прогресу (`user_results`).

## 🛠 Технології

- **Фреймворк:** [React](https://react.dev/) (JavaScript)
- **Збірка:** [Vite](https://vitejs.dev/)
- **База даних:** Firebase Firestore (хмарне зберігання історії користувачів) + Firebase Auth
- **Стилізація:** CSS3 (CSS Variables, Flexbox, Glassmorphism)
- **Деплой:** Netlify

## 🚀 Як запустити локально

1. Клонувати репозиторій:

   ```bash
   git clone git@github.com:noodlex3m/beb-quiz.git
   ```

2. Встановити залежності:

   ```bash
   npm install
   ```

3. Запустити локальний сервер розробки:
   ```bash
   npm run dev
   ```
